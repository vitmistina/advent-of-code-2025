# API Contract: Day 3 Part 2

## Endpoint

- **Function**: `solve_part2(input_text: str) -> int`

## Input

- `input_text`: str (multi-line string, each line is a bank)

## Output

- Returns: int (sum of max 12-digit numbers from each bank)

## Example

```
Input:
987654321111111
811111111111119
234234234234278
818181911112111

Output: 3121910778619
```
